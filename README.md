ShanHiFormRules 主要提供以下能力：

- 支持注解方式执行参数检验（类似ElementUI-validator风格）。

- 提供较为常用的规则实现，同时提供了自定义规则扩展能力，

  告别Hibernate validator 分组方式无法高度定制多API共享入参的情况。

- 更多规则可以自己提PR

使用说明：https://seemountainsea.github.io/formrules/
